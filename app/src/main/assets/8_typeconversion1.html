Implicit Type Conversion

Python automatically converts one data type to another when it's required. This happens during operations between different data types, and Python handles it behind the scenes.

Example 
# Implicit conversion of int to float
x = 10      
y = 3.14    
result = x + y
print(result)  # Output: 13.14

# Implicit conversion of int to complex
z = 5 + 0j
result_complex = x + z
print(result_complex)

In this example, the integer x is implicitly converted to a float and a complex number during the addition operations.


Explicit Type Conversion

Explicit type conversion, also known as type casting, is done by the programmer using predefined functions to change the data type of an object.

There are several built-in functions for explicit type conversion.

  • int(): Converts a number or a string to an integer.
  • float(): Converts a number or a string to a floating-point number.
  • str(): Converts an object to a string.
  • list(), tuple(), set(): Convert other iterable objects to lists, tuples, or sets.
  • bool(): Converts a value to a Boolean.

It's important to note that not all type conversions are valid or result in meaningful values. For example, converting a string containing letters to an integer might raise a ValueError.

Example 
float_number = 3.99
int_number = int(float_number)
print(int_number)  

num = 42
str_num = str(num)
print(str_num)  

num_str = "3.14"
float_num = float(num_str)
print(float_num) 

my_list = [1, 2, 3]
my_tuple = tuple(my_list)
print(my_tuple)  

invalid_str = "abc"
try:
    invalid_int = int(invalid_str)
except ValueError as e:
    print(f"Error: {e}")

In this example, attempting to convert a string containing letters to an integer raises a ValueError. It's always a good practice to handle potential errors when performing explicit type conversion.


Understanding and appropriately using type conversion is crucial for writing robust and flexible code in Python. Always be mindful of the data types involved in your operations to avoid unexpected behavior or errors.
