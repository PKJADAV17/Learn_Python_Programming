Default constructor

In Python, the default constructor is a special method named __init__ that takes no parameters or takes only the self parameter. It is automatically called when an object is created from a class and is used to initialize the attributes of the object or perform any setup that is common to all objects of the class.

Here's an example of a class with a default constructor.

Example 
class MyClass:
    def __init__(self):
        self.default_attribute = "default_value"
        print("Default constructor called")

my_object = MyClass()

  • In this example __init__(self) is the default constructor method.
  • self is a reference to the instance of the class (the object being created).
  • Inside the constructor, self.default_attribute is an attribute of the object, and its value is initialized to "default_value."

When you create an object of the class (my_object in this case), the __init__ method is automatically called, and you can see the output "Default constructor called" printed.

The default constructor is optional. If you don't define one in your class, Python provides a default constructor that takes no parameters and doesn't perform any specific initialization. However, if you want to customize the initialization process, you can define your own default constructor in the class.


Parameterized Constructor

In Python, a parameterized constructor is a special method named __init__ that takes parameters in addition to the self parameter. This allows you to pass values to the constructor when creating an object, and these values are used to initialize the attributes of the object.

Here's an example of a class with a parameterized constructor.

Example 
class MyClass:
    def __init__(self, parameter1, parameter2):
        self.attribute1 = parameter1
        self.attribute2 = parameter2
        print("Parameterized constructor called with parameters:", parameter1, parameter2)

my_object = MyClass("value1", "value2")
Output

  • In this example __init__(self, parameter1, parameter2) is the parameterized constructor method.
  • self is a reference to the instance of the class (the object being created).
  • parameter1 and parameter2 are parameters that you can pass when creating an object of the class.
  • Inside the constructor, self.attribute1 and self.attribute2 are attributes of the object, and their values are initialized with the values of the parameters.

When you create an object of the class (my_object in this case), you pass specific values ("value1" and "value2" in this example) as arguments, and the __init__ method is automatically called with those values. The output will be "Parameterized constructor called with parameters: value1 value2".

Parameterized constructors are useful when you want to customize the initialization of objects based on the values provided during object creation.


Constructors are commonly used to set up the initial state of an object, and they provide a way to pass initial values to the attributes of the object.
