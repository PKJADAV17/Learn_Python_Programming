Identity Operators

Identity operators in Python are used to compare the memory locations of two objects. These operators check whether two variables refer to the same object in memory or not. There are two identity operators in Python: is and is not.

Here's a table summarizing the identity operators in Python.

OperatorDescriptionExampleResult
isIdentitya is bTrue if a is the same object as b
is notNegated identitya is not bTrue if a is not the same object as b

These identity operators are often used to compare objects in a more specific way than the equality operators (== and !=). While equality checks whether the values of two objects are the same, identity checks whether the two variables refer to the exact same object in memory.


Example 
a = [1, 2, 3]
b = a
c = [1, 2, 3]

print(a is b)    
print(a is c)    

print(a is not b)
print(a is not c)

Membership Operators

Membership operators in Python are used to test whether a value is a member of a sequence, such as a string, list, tuple, or set. The two membership operators in Python are in and not in. These operators return a Boolean value (True or False) based on whether the specified value is found in the sequence or not.

Here's a table summarizing the membership operators in Python.

OperatorDescriptionExampleResult
inMembershipvalue in sequenceTrue if value is found in the sequence
not inNegated Membershipvalue not in sequenceTrue if value is not found in the sequence

Membership operators are commonly used in conditional statements and loops to check if a specific value is present in a given sequence.


Example 
my_list = [1, 2, 3, 4, 5]
my_tuple = ('apple', 'banana', 'orange')
my_string = "Hello, World!"
my_set = {1, 2, 3, 4, 5}

print(3 in my_list)

print('pear' not in my_tuple)

Bitwise Operators

Bitwise operators in Python perform operations on the individual bits of binary representations of integers. These operators are often used in low-level programming, such as in working with hardware, graphics, encryption, and other areas where bitwise manipulation is required.

Here's a table summarizing the bitwise operators in Python.

Operator Name Description
&  AND Sets each bit to 1 if both bits are 1
| OR Sets each bit to 1 if one of two bits is 1
^ XOR Sets each bit to 1 if only one of two bits is 1
~ NOT Inverts all the bits
<< Zero fill left shift Shift left by pushing zeros in from the right and let the leftmost bits fall off
>> Signed right shift Shift right by pushing copies of the leftmost bit in from the left, and let the rightmost bits fall off

Bitwise operators work on integer values, and the results are integers as well. Understanding binary representation and bitwise operations is crucial when working with these operators.


Example 
a = 15   
b = 7    

and_result = a & b  
or_result = a | b   
xor_result = a ^ b  

not_a = ~a          
left_shift_result = a << 1   
right_shift_result = a >> 1  

print(and_result, or_result, xor_result, not_a, left_shift_result, right_shift_result)
