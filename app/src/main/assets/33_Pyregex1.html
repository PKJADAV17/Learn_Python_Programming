[] A set of characters "[a-m]"

Special Sequences

Certainly! Special sequences in regular expressions are sequences of characters that have a special meaning. Here's a table with some common special sequences and their meanings in regular expressions.

Character Description Example
\A Returns a match if the specified characters are at the beginning of the string "\AThe"
\b Returns a match where the specified characters are at the beginning or at the end of a word
(the "r" in the beginning is making sure that the string is being treated as a "raw string")		 r"\bain"
r"ain\b"
\B Returns a match where the specified characters are present, but NOT at the beginning (or at the end) of a word
(the "r" in the beginning is making sure that the string is being treated as a "raw string")		 r"\Bain"
r"ain\B"
\d Returns a match where the string contains digits (numbers from 0-9) "\d"
\D Returns a match where the string DOES NOT contain digits "\D"
\s Returns a match where the string contains a white space character "\s"
\S Returns a match where the string DOES NOT contain a white space character "\S"
\w Returns a match where the string contains any word characters (characters from a to Z, digits from 0-9, and the underscore _ character) "\w"
\W Returns a match where the string DOES NOT contain any word characters "\W"
\Z Returns a match if the specified characters are at the end of the string "Spain\Z"

Sets

In regular expressions, character sets, also known as character classes, are used to specify a set of characters to match at a particular position in the input. Here's a table with some common character sets and their meanings.

Set Description
[arn] Returns a match where one of the specified characters (a, r, or n) is present
[a-n] Returns a match for any lower case character, alphabetically between a and n
[^arn] Returns a match for any character EXCEPT a, r, and n
[0123] Returns a match where any of the specified digits (0, 1, 2, or 3) are present
[0-9] Returns a match for any digit between 0 and 9
[0-5][0-9] Returns a match for any two-digit numbers from 00 and 59
[a-zA-Z] Returns a match for any character alphabetically between a and z, lower case OR upper case
[+] In sets, +, *, ., |, (), $,{ } has no special meaning, so [+] means: return a match for any + character in the string

Match Object

In Python's re module, when you use functions like match(), search(), or findall(), they return a match object if a match is found. The match object provides information about the match and has methods to retrieve details about the match.

Here are some common methods of a match object.

  • group([group1, ...]): Returns one or more subgroups of the match. If no argument is given, it returns the entire match.
  • start([group]) and end([group]): Returns the start and end positions of the match or a specified group.
  • span([group]): Returns a tuple containing the (start, end) positions of the match or a specified group.
  • groups([default]): Returns a tuple containing all the subgroups of the match.
Example 
import re

pattern = re.compile(r'(\d{2})-(\d{2})-(\d{4})')
match = pattern.match('28-11-2023')

print(match.group())   
print(match.group(1))  
print(match.group(2))  
print(match.group(3))  

print(match.start())   
print(match.end())     
print(match.start(2))  
print(match.end(2))    

print(match.span())    
print(match.span(1))   

print(match.groups())

These methods allow you to access various details about the matched text and its subgroups. Keep in mind that if there is no match, these methods may raise an AttributeError, so it's a good practice to check if a match is found before attempting to access the match object's properties.


search Function

The re.search() function in Python is used to search for a pattern in a string. It scans the entire string and returns a match object if a match is found. If no match is found, it returns None. The re.search() function is less restrictive than re.match(), as it searches for the pattern anywhere in the string, not just at the beginning.

Here's a simple example demonstrating the use of re.search().

Example 
import re

pattern = re.compile(r'\d{3}')  

text = "The price is 123 dollars."

match = pattern.search(text)

if match:
    print("Match found:", match.group())
else:
    print("No match found.")
Output
Click on try it to see output

In this example, the pattern \d{3} is looking for three consecutive digits. The re.search() function finds the first occurrence of this pattern in the given text, and if a match is found, it prints the matched string using match.group().


You can also use the start(), end(), and span() methods of the match object returned by re.search() to get information about the position of the match in the string.

Example 
import re

pattern = re.compile(r'\d{3}')  

text = "The price is 123 dollars."

match = pattern.search(text)

if match:
    print("Match found:", match.group())
else:
    print("No match found.")

if match:
    print("Start position:", match.start())
    print("End position:", match.end())
    print("Match span:", match.span())
Output

This allows you to obtain the start and end positions of the match and the span as a tuple.

Keep in mind that re.search() stops as soon as it finds the first match. If you need to find all occurrences of a pattern in a string, you might want to use re.finditer() or re.findall() instead.


split Function

The re.split() function in Python is used to split a string based on a specified pattern. It takes a regular expression pattern as its first argument and the target string as the second argument. It returns a list of substrings obtained by splitting the original string at occurrences of the specified pattern.

Here's a simple example demonstrating the use of re.split().

Example 
import re

pattern = re.compile(r'\s+')

text = "This is a sample   string with multiple    spaces."

result = pattern.split(text)

print(result)
Output
Click on try it to see output

In this example, the pattern \s+ represents one or more whitespace characters. The re.split() function splits the given text wherever it finds one or more consecutive whitespace characters.


You can use various regular expression patterns in re.split() to split a string based on different criteria, such as specific characters, patterns, or even groups.

Here's another example where the string is split based on either a comma or a semicolon.

Example 
import re

pattern = re.compile(r'[,;]')

text = "apple,orange;banana;grape"

result = pattern.split(text)

print(result)
Output
Click on try it to see output

In this case, the pattern [,;] matches either a comma or a semicolon, causing the string to be split at these occurrences.

Keep in mind that the re.split() function is useful for more complex splitting scenarios where the built-in str.split() method might not be sufficient.


sub Function

The re.sub() function in Python is used to replace occurrences of a pattern in a string with a specified replacement string. It takes three main arguments: the pattern to search for, the replacement string, and the target string. Optionally, you can provide additional arguments to control the number of substitutions or modify the behavior.

Here's a basic example demonstrating the use of re.sub().

Example 
import re

pattern = re.compile(r'\d+')  

text = "The price is 123 dollars."

result = pattern.sub("X", text)

print(result)
Output
Click on try it to see output

In this example, the pattern \d+ is used to match one or more digits. The re.sub() function replaces all occurrences of digits in the text with the letter "X".


You can also use a function as the replacement argument. The function takes a match object as an argument and returns the replacement string. This allows for more dynamic replacements based on the matched content.

Example 
import re

def square(match):
    num = int(match.group())
    return str(num * num)

pattern = re.compile(r'\d+')  

text = "The squares are 1, 4, 9, and 16."

result = pattern.sub(square, text)

print(result)
Output
Click on try it to see output

In this example, the square function takes a match object, extracts the matched digits, squares the number, and returns the result.

The re.sub() function is quite flexible and useful for performing complex replacements in strings based on regular expressions.


findall Function

The re.findall() function in Python is used to find all occurrences of a pattern in a string and return them as a list. It takes two main arguments: the pattern to search for and the target string. Optionally, you can provide additional arguments to modify the behavior.

Here's a basic example demonstrating the use of re.findall().

Example 
import re

pattern = re.compile(r'\d+')  

text = "There are 123 apples and 456 oranges in the basket."

result = pattern.findall(text)

print(result)
Output
Click on try it to see output

In this example, the pattern \d+ is used to match one or more digits. The re.findall() function finds all occurrences of digits in the text and returns them as a list.


You can use more complex patterns to match specific content in the string.

Example 
import re

pattern = re.compile(r'\b\w{3}\b')  

text = "The cat is on the mat."

result = pattern.findall(text)

print(result)
Output
Click on try it to see output

In this example, the pattern \b\w{3}\b is used to match three-letter words. The \b represents a word boundary, and \w{3} matches exactly three word characters.

The re.findall() function is particularly useful when you want to extract specific patterns or content from a string and obtain all occurrences.
