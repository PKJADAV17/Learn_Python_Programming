• __iter__() method:
    • This method returns the iterator object itself. It is called when you use the iter() function on an object or when you use it in a for loop.
  • __next__() method:
    • This method is responsible for returning the next element in the iterator. When there are no more items to return, it should raise the StopIteration exception to signal the end of the iteration.

Here's a simple example of an iterator in Python.

Example 
class MyIterator:
    def __iter__(self):
        self.counter = 0
        return self

    def __next__(self):
        if self.counter < 5:
            result = self.counter
            self.counter += 1
            return result
        else:
            raise StopIteration

my_iter = MyIterator()
for num in my_iter:
print(num)

In this example, MyIterator is a simple iterator that produces numbers from 0 to 4. When you use it in a for loop, it calls the __iter__() method to get the iterator object and then repeatedly calls __next__() until StopIteration is raised.


Python provides a convenient way to create iterators using the iter() function and the next() function.

Example 
my_iter = iter([0, 1, 2, 3, 4])
print(next(my_iter))  
print(next(my_iter))

Many built-in objects in Python are iterable, meaning you can loop through their elements using a for loop or the iter() and next() functions. Examples include lists, tuples, dictionaries, and strings.
