Numeric Types

Numeric types in Python represent numbers and include integers (int), floating-point numbers (float), and complex numbers (complex).

  • int: Integer data type represents whole numbers without any decimal points. Example: x = 5
  • float: Float data type represents real numbers with decimal points. Example: y = 3.14
  • complex: Complex numbers have a real and an imaginary part. Example: z = 2 + 3j
Example 
# Example of Numeric Types
x = 5       # int
y = 3.14    # float
z = 2 + 3j  # complex

Sequence Types

Sequence types in Python represent ordered collections of elements. The primary sequence types are strings (str), lists (list), tuples (tuple) type.

  • str (String)

    Represents sequences of characters. Strings are immutable (cannot be changed after creation). Example: name = "John".

    name = "John"
  • list

    Represents a mutable ordered sequence of elements. Elements can be of different data types. Example: numbers = [1, 2, 3, 4].

    numbers = [1, 2, 3, 4]
  • tuple

    Represents an immutable ordered sequence of elements. Once created, elements cannot be changed. Example: coordinates = (3, 4)

    coordinates = (3, 4)

Set Types

In Python, set types represent unordered collections of unique elements. There are two primary set types: set and frozenset.

  • set

    Sets are mutable collections of unique elements. They are unordered, meaning that the order of elements is not guaranteed. Example: unique_numbers = {1, 2, 3, 4}

    unique_numbers = {1, 2, 3, 4}
  • frozenset

    Frozensets are immutable collections of unique elements. Once created, their elements cannot be changed. Example: immutable_set = frozenset({1, 2, 3, 4})

    immutable_set = frozenset({1, 2, 3, 4})

Mapping Type

In Python, the primary mapping type is the dictionary (dict). Dictionaries are mutable and represent an unordered collection of key-value pairs, where each key must be unique.


dict (Dictionary)

Dictionaries are mutable collections that map keys to values. They are unordered, meaning that the order of elements is not guaranteed. Keys must be immutable (strings, numbers, or tuples) while values can be of any data type. Example: person = {"name": "John", "age": 30, "city": "New York"}

person = {"name": "John", "age": 30, "city": "New York"}

Boolean Type

In Python, the Boolean type (bool) is used to represent logical values. There are only two Boolean values: True and False. Booleans are essential for making decisions in code and controlling the flow of a program through conditional statements.

Booleans represent the truth values True or False.

is_true = True
is_false = False

These data types are the building blocks for constructing more complex data structures and performing various operations in Python programming. Understanding them is crucial for effective coding and problem-solving.
