for loop

The for loop in Python is used for iterating over a sequence (that can be a list, tuple, string, or other iterable objects) and executing a block of code for each element in the sequence.

Syntax 
for variable in sequence:
    # code to be executed for each item in the sequence
  • variable: This is a variable that takes on the value of each item in the sequence during each iteration of the loop.
  • sequence: This is the sequence of elements over which the loop iterates.

Here is an example using a list of numbers.

Example 
numbers = [1, 2, 3, 4, 5]

for num in numbers:
    print(num)
Output

In this example, the for loop iterates over the list of numbers, and the variable num takes on each value in the list in turn. The print(num) statement is executed for each number in the list.


Range Function

The range() function is often used with for loops to generate a sequence of numbers.

Example 
for i in range(5):
    print(i)
Output
Click on try it to see output

In this example, range(5) generates a sequence of numbers from 0 to 4. The for loop then iterates over this sequence, and the variable i takes on each value in the sequence.


while loop

The while loop is used to repeatedly execute a block of code as long as a given condition is true.

Syntax 
while condition:
    # code to be executed as long as the condition is true

condition: A boolean expression. The loop continues to execute as long as this condition evaluates to True.


Here's a simple example where a while loop is used to print numbers from 0 to 4.

Example 
count = 0
while count < 5:
    print(count)
    count += 1
Output
Click on try it to see output
  • In this example The count variable is initialized to 0.
  • The while loop continues to execute as long as count is less than 5.
  • Inside the loop, the current value of count is printed, and then count is incremented by 1.

Infinite while Loop

Be cautious when using while loops to avoid unintentional infinite loops. An infinite loop occurs when the loop condition always evaluates to True, and the loop never exits.

Example 
# Infinite loop
while True:
    print("This is an infinite loop!")

To break out of an infinite loop, you can use the break statement.


else Statement in loops

In Python, the else clause in loops is a feature that might seem a bit counterintuitive at first, but it provides additional functionality to loops, both for and while. The else clause in a loop is executed when the loop condition becomes false. However, there is an exception: if the loop is terminated by a break statement, the else clause is skipped.

Here's the basic syntax

Syntax of for Loop with else 
for variable in sequence:
    # code to be executed for each item in the sequence
else:
    # code to be executed when the loop completes normally (i.e., without a break)

Syntax while Loop with else 
while condition:
    # code to be executed as long as the condition is true
else:
    # code to be executed when the loop condition becomes false (i.e., without a break)

Example of else statement with for Loop

Example 
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
else:
    print("Loop finished.")
Output

In this example, the else block is executed after the for loop completes its iterations over the list of fruits. If the loop completes normally without encountering a break statement, the else block is executed and "Loop finished." is printed.


Example of else statement with while Loop

Example 
count = 0
while count < 5:
    print(count)
    count += 1
else:
    print("Loop finished.")
Output
Click on try it to see output

In this example, the else block is executed when the condition count < 5 becomes false. Again, if the loop completes normally without a break, the else block is executed.

Remember that the else clause in loops is optional, and you don't have to include it if you don't need it for your specific use case.
