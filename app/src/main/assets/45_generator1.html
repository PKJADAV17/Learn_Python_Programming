Let's go through a practical example of using a generator. Suppose you want to generate the squares of numbers up to a given limit.

Here's how you could implement it using a generator.

Example 
def square_generator(limit):
    n = 0
    while n < limit:
        yield n ** 2
        n += 1

limit = 5
squares = square_generator(limit)

for square in squares:
print(square)

In this example, the square_generator function is a generator that yields the square of each number from 0 to limit - 1. The generator is used in a for loop to iterate over the generated values, printing each square.

The generator produces each square on-the-fly without having to store all the values in memory. This can be especially useful if you were working with a very large limit where storing all the squares in memory might be impractical.


You can also use the next() function to manually retrieve values from the generator.

Example 
def square_generator(limit):
    n = 0
    while n < limit:
        yield n ** 2
        n += 1

limit = 5
squares = square_generator(limit)

print(next(squares))  
print(next(squares))  
print(next(squares))  
print(next(squares))
print(next(squares))
