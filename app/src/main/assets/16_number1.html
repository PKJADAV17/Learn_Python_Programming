  • Integers (int):
    • Integers are whole numbers without any decimal point.
    • Example: 0, 42, -17
    • In Python, integers can be of arbitrary precision, meaning they can be as large as your computer's memory allows.
  • Floating-Point Numbers (float):
    • Floating-point numbers, or floats, are numbers that have a decimal point or are expressed using scientific notation.
    • Example: 3.14, -0.001, 2.5e-3 (scientific notation for 0.0025)
    • Floating-point numbers in Python are implemented using the IEEE 754 standard.
  • Complex Numbers (complex):
    • Complex numbers have a real and an imaginary part. They are represented in the form a + bj, where a is the real part, b is the imaginary part, and j is the imaginary unit (equal to the square root of -1).
    • Example: 2 + 3j, -1 - 0.5j
    • Complex numbers can be used for various mathematical operations, including those involving square roots of negative numbers.

Operations on Numbers

Python supports various operations on numbers, including.

  • Arithmetic Operations:
    • Addition (+), Subtraction (-), Multiplication (*), Division (/), Floor Division (//), Modulus (%), and Exponentiation (**).
  • Comparison Operations:
    • Equal (==), Not Equal (!=), Greater Than (>), Less Than (<), Greater Than or Equal To (>=), Less Than or Equal To (<=).
  • Logical Operations:
    • and, or, not for combining multiple conditions.

Type Conversion

You can convert numbers from one type to another using functions like int(), float(), and complex().

Example 
# Type conversion examples
a = 5
b = float(a)  # Convert integer to float
c = complex(a)  # Convert integer to complex
