Numeric Literals
  • Integer Literals: These are whole numbers without any decimal point. Example: 42, -10, 0.
  • Floating-point Literals: These are numbers with a decimal point or in exponential form. Example: 3.14, 2.5e2.
  • Complex Literals: These are written with a real and imaginary part, expressed as real + imaginaryj. Example: 1 + 2j.
Example 
# Numeric literals
integer_literal = 42
float_literal = 3.14
complex_literal = 1 + 2j

String Literals
  • Strings are sequences of characters enclosed in single (') or double (") quotes.
  • Triple-quoted strings (''' or """) can be used for multiline strings.
Example 
# String literals
single_quoted = 'Hello, World!'
double_quoted = "Python is great!"
multiline_string = '''This is a
multiline
string.'''

Boolean Literals

These represent the two Boolean values: True and False.

Example 
# Boolean literals
true_literal = True
false_literal = False

None Literal

The None literal represents the absence of a value or a null value.

Example 
# None literal
none_literal = None

Sequence Type Literals

Lists, tuples, and ranges are literals representing sequence types.

Example 
# List and tuple literals
list_literal = [1, 2, 3]
tuple_literal = (4, 5, 6)

Mapping Type Literals

Dictionary literals represent mapping types.

Example 
# Dictionary literal
dict_literal = {'key1': 'value1', 'key2': 'value2'}

Set Literals

Sets and frozensets can be represented using set literals.

Example 
# Set and frozenset literals
set_literal = {1, 2, 3}
frozenset_literal = frozenset([4, 5, 6])

Binary Literals (introduced in Python 2.6 and 3.0):

Binary literals represent numbers in binary form using the prefix 0b or 0B.

Example 
# Binary literal
binary_literal = 0b10101

These are the main types of literals in Python. They are used to represent constant values within your code. Literals provide a convenient way to express values in a readable and straightforward manner.
