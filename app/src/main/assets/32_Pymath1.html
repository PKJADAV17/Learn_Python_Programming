Mathematical Constants

To use the functions and constants provided by the math module, you need to import it first. You can do this using the import statement.

The math module provides some mathematical constants, such as.

  • math.pi: The mathematical constant π (pi).
  • math.e: The mathematical constant e.

Here's an example demonstrating the use of mathematical constants from the math module in Python.

Example 
import math

print("Value of pi (π):", math.pi)
print("Value of e:", math.e)

radius = 3
circumference = 2 * math.pi * radius
print("Circumference of the circle with radius", radius, "is:", circumference)

radius = 4
area = math.pi * math.pow(radius, 2)
print("Area of the circle with radius", radius, "is:", area)

Basic Mathematical Operations

The math module includes functions for basic mathematical operations, including.

  • math.sqrt(x): Returns the square root of x.
  • math.pow(x, y): Returns x raised to the power y.
  • math.exp(x): Returns the exponential of x (e^x).
  • math.log(x): Returns the natural logarithm of x.
  • math.log10(x): Returns the base-10 logarithm of x.

Here's an example demonstrating basic mathematical operations using functions from the math module in Python.

Example 
import math

x = 16
y = 4

sqrt_result = math.sqrt(x)
print(f"Square root of {x} is: {sqrt_result}")

power_result = math.pow(x, y)
print(f"{x} raised to the power {y} is: {power_result}")

exp_result = math.exp(x)
print(f"Exponential of {x} is: {exp_result}")

log_result = math.log(x)
print(f"Natural logarithm of {x} is: {log_result}")

log10_result = math.log10(x)
print(f"Base-10 logarithm of {x} is: {log10_result}")

  • Calculating the square root of x using math.sqrt.
  • Raising x to the power of y using math.pow.
  • Calculating the exponential of x using math.exp.
  • Finding the natural logarithm of x using math.log.
  • Calculating the base-10 logarithm of x using math.log10.

Trigonometric Functions

The math module provides various trigonometric functions, such as.

  • math.sin(x), math.cos(x), math.tan(x): Sine, cosine, and tangent of x (in radians).
  • math.asin(x), math.acos(x), math.atan(x): Arcsine, arccosine, and arctangent of x, returning results in radians.

Here's an example demonstrating the use of trigonometric functions from the math module in Python.

Example 
import math

angle_in_degrees = 45

angle_in_radians = math.radians(angle_in_degrees)

sin_value = math.sin(angle_in_radians)
cos_value = math.cos(angle_in_radians)
tan_value = math.tan(angle_in_radians)

print(f"Sine of {angle_in_degrees} degrees is: {sin_value}")
print(f"Cosine of {angle_in_degrees} degrees is: {cos_value}")
print(f"Tangent of {angle_in_degrees} degrees is: {tan_value}")

asin_value = math.degrees(math.asin(sin_value))
acos_value = math.degrees(math.acos(cos_value))
atan_value = math.degrees(math.atan(tan_value))

print(f"Arcsine of sin({angle_in_degrees} degrees) is: {asin_value}")
print(f"Arccosine of cos({angle_in_degrees} degrees) is: {acos_value}")
print(f"Arctangent of tan({angle_in_degrees} degrees) is: {atan_value}")

  • We then use trigonometric functions like math.sin, math.cos, and math.tan to calculate the sine, cosine, and tangent of the angle in radians.
  • Additionally, we use the inverse trigonometric functions (math.asin, math.acos, math.atan) to find the angles (in degrees) corresponding to the sine, cosine, and tangent values calculated earlier.

Angular Conversion

There are functions to convert angles between degrees and radians.

  • math.degrees(x): Converts angle x from radians to degrees.
  • math.radians(x): Converts angle x from degrees to radians.

Here's an example demonstrating angular conversion functions in the math module in Python.

Example 
import math

angle_in_degrees = 60

angle_in_radians = math.radians(angle_in_degrees)
print(f"{angle_in_degrees} degrees is equivalent to {angle_in_radians} radians")

angle_in_degrees_again = math.degrees(angle_in_radians)
print(f"{angle_in_radians} radians is equivalent to {angle_in_degrees_again} degrees")

  • We then print the equivalent angle in radians.
  • Next, we use math.degrees to convert the radians back to degrees.
  • Finally, we print the equivalent angle in degrees.

Hyperbolic Functions

The math module includes hyperbolic functions like math.sinh(x), math.cosh(x), and math.tanh(x).

Here's an example demonstrating the use of hyperbolic functions from the math module in Python.

Example

sinh_value = math.sinh(x)

print(f"Hyperbolic cosine of {x} is: {cosh_value}")
print(f"Hyperbolic tangent of {x} is: {tanh_value}")
