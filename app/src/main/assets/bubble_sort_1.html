Example

Let us take an example of the following elements.

5 1 4 2 8

Here apply the bubble sort algorithm to sort the elements.

Example
def bubble_sort(list):
    for j in range(len(list)-1):
        for i in range(j):
            if list[i]>list[i+1]:
                temp = list[i]
                list[i] = list[i+1]
                list[i+1] = temp

list = [1,4,2,5,8]
bubble_sort(list)
print(list)
