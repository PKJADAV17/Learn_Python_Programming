Creating a Set

In Python, you can create a set using either curly braces {} or the set() constructor.


Using Curly Braces

Here is the example of creating a set using curly braces.

Example 
my_set = {1, 2, 3, 4, 5}
print(my_set)
Output
Click on try it to see output

In this example, my_set is a set containing the elements 1, 2, 3, 4, and 5.

If you want to create an empty set, you should use the set() constructor because an empty pair of curly braces {} will create an empty dictionary, not an empty set.


Using the set() Constructor

Here is the example of creating a set using set() constructor.

Example 
another_set = set([2, 4, 6, 8, 10])
print(another_set)
Output
Click on try it to see output

Here, set() is a constructor that can be used to create a set. You can pass an iterable (like a list) to the set() constructor, and it will create a set containing the unique elements of that iterable. In this case, another_set is a set containing the elements 2, 4, 6, 8, and 10.


Adding Elements in set

In Python, you can add elements to a set using the add() method.

Example 
my_set = {1, 2, 3, 4, 5}

my_set.add(6)
print(my_set)

my_set.update([7, 8, 9])
print(my_set)
Output
Click on try it to see output

The add() method is used to add a single element to the set. If you want to add multiple elements, you can use the update() method and pass an iterable (e.g., a list or another set) containing the elements you want to add.

It's important to note that sets do not allow duplicate elements. If you try to add an element that already exists in the set, it won't create a duplicate.


Removing Elements from set

In Python, you can remove elements from a set using the remove() or discard() methods.


remove() Method

The remove() method removes the specified element from the set. If the element is not present, it raises a KeyError.

Example

my_set.remove(3)
print(my_set)

Set Operation

Sets in Python support various mathematical operations that you can perform on them.


Union

The union of two sets contains all unique elements from both sets.

Example 
set1 = {1, 2, 3}
set2 = {3, 4, 5}

union_set = set1.union(set2)
print(union_set)

Intersection

The intersection of two sets contains elements that are common to both sets.

Example 
set1 = {1, 2, 3}
set2 = {3, 4, 5}

intersection_set = set1.intersection(set2)
print(intersection_set)

The symmetric difference of two sets contains elements that are in either of the sets, but not in both.

Example

Subset and Superset

In Python, you can check whether one set is a subset or superset of another set using the issubset() and issuperset() methods, respectively.

Example 
set1 = {1, 2}
set2 = {1, 2, 3, 4}

is_subset = set1.issubset(set2)
print(is_subset)

is_superset = set1.issuperset(set2)
print(is_superset)
Output
Click on try it to see output

In this example, set1 is a subset of set2 because all elements of set1 are also present in set2, and set1 is not a superset of set2 because all elements of set2 are not present in set1.


You can also use the subset and superset operators <= and >=

Example 
set1 = {1, 2}
set2 = {1, 2, 3, 4}

is_subset = set1 <= set2
print(is_subset)

is_superset = set2 >= set1
print(is_superset)
Output
Click on try it to see output

These operators provide a more concise way to express subset and superset relationships.


Keep in mind that a set is considered a subset or superset of itself. If you want to check for a proper subset or superset (i.e., one set being a subset or superset but not equal), you can use the < and > operators.

Example 
set1 = {1, 2}
set2 = {1, 2, 3, 4}

is_proper_subset = set1 < set2
print(is_proper_subset)

is_proper_superset = set2 > set1
print(is_proper_superset)
Output
Click on try it to see output

These operations are useful for comparing the relationships between sets in terms of their elements.


Frozen Sets

In Python, a frozenset is a built-in set that is immutable, meaning its elements cannot be changed or modified after the frozenset is created. Unlike a regular set, a frozenset is hashable and can, therefore, be used as a key in a dictionary or as an element in another set.

Here's how you can create a frozenset.

Example 
my_frozenset = frozenset([1, 2, 3, 4, 5])
print(my_frozenset)
Output
Click on try it to see output

Once a frozenset is created, you cannot add or remove elements from it. Attempting to do so will result in an AttributeError.


However, you can perform non-modifying operations such as union, intersection, difference, and symmetric difference, similar to regular sets

Example 
frozenset1 = frozenset([1, 2, 3])
frozenset2 = frozenset([3, 4, 5])

union_frozenset = frozenset1 | frozenset2
intersection_frozenset = frozenset1 & frozenset2
difference_frozenset = frozenset1 - frozenset2
symmetric_difference_frozenset = frozenset1 ^ frozenset2

print(union_frozenset)
print(intersection_frozenset)
print(difference_frozenset)
print(symmetric_difference_frozenset)
Output
Click on try it to see output

frozenset objects are particularly useful in situations where you need an immutable set, for example, when you want to use sets as keys in dictionaries or elements in other sets. Regular sets are mutable and cannot be used as dictionary keys or elements in other sets.
