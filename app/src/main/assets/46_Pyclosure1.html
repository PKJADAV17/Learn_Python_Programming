`def outer_function(x):
    def inner_function(y):
        return x + y

    return inner_function

closure_instance = outer_function(10)

result = closure_instance(5)

print(result)

In this example, outer_function takes an argument x and defines an inner function inner_function that takes another argument y. The inner function, when returned, forms a closure that "remembers" the value of x from the enclosing scope. When you later call closure_instance(5), it adds 5 to the remembered value of x (which was 10), resulting in 15.

Closures are useful in situations where you want to create functions with behavior dependent on some external state, and you want that state to persist across multiple function calls. They are commonly used in functional programming and can be a powerful tool for creating more flexible and modular code.
