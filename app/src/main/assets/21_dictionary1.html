Features of Python dictionaries
  • Mutable: Dictionaries in Python are mutable, meaning you can modify them by adding, removing, or updating key-value pairs.
  • Unordered: Dictionaries are unordered collections, which means they do not maintain any specific order of elements. This is different from sequences like lists or tuples where the order is important.
  • Dynamic: You can add, modify, or remove key-value pairs from a dictionary after it has been created.

Here's a basic example of a dictionary.

Example 
my_dict = {'key1': 'value1', 'key2': 'value2', 'key3': 'value3'}
print(my_dict)

  • 'value1', 'value2', and 'value3' are the corresponding values.

Accessing Dictionary Values

Accessing values in a Python dictionary is done by providing the key associated with the value you want to retrieve. There are several ways to access dictionary values.


get() Method

You can use square brackets [] with the key inside to access the corresponding value.

The get() method allows you to retrieve the value for a given key. If the key is not present, it returns None (or a default value you can specify).

Example 
my_dict = {'key1': 'value1', 'key2': 'value2', 'key3': 'value3'}

value_of_key1 = my_dict['key1']
print(value_of_key1)  

value_of_key2 = my_dict.get('key2')
print(value_of_key2)  

value_of_key4 = my_dict.get('key4', 'default_value')
print(value_of_key4)
Output
Click on try it to see output

Using get() is often preferable if you're not sure whether the key exists, as it won't raise a KeyError if the key is not found.


Using a Loop to Iterate Over Keys

You can also iterate over the keys and access the values in a loop.

Example 
my_dict = {'key1': 'value1', 'key2': 'value2', 'key3': 'value3'}
for key in my_dict:
    print(f'{key}: {my_dict[key]}')
Output

Using the items() Method

The items() method returns a view of the dictionary's key-value pairs as tuples. This allows you to iterate over both keys and values.

Example 
my_dict = {'key1': 'value1', 'key2': 'value2', 'key3': 'value3'}
for key, value in my_dict.items():
    print(f'{key}: {value}')

These methods provide different ways to access values in a dictionary based on your specific needs. Choose the method that fits the context of your code.


Update Dictionary

Updating a dictionary in Python involves adding, modifying, or removing key-value pairs. Here are several ways to update a dictionary.


Adding or Modifying Key-Value Pairs

You can add a new key-value pair or modify the value of an existing key.

Example 
my_dict = {'key1': 'value1', 'key2': 'value2', 'key3': 'value3'}

my_dict['key4'] = 'value4'
my_dict['key2'] = 'new_value'
print(my_dict)
Output
Click on try it to see output

update() Method

The update() method allows you to merge the contents of another dictionary or an iterable of key-value pairs into the current dictionary.

Example 
my_dict = {'key1': 'value1', 'key2': 'value2', 'key3': 'value3'}
new_data = {'key4': 'value4', 'key5': 'value5'}

my_dict.update(new_data)

print(my_dict)
Output
Click on try it to see output

The updated dictionary will include the new key-value pairs.


setdefault()

The setdefault() method sets the value of a key if the key is not already in the dictionary.

Example 
my_dict = {'key1': 'value1', 'key2': 'value2', 'key3': 'value3'}

# Setting a default value for a new key
my_dict.setdefault('key4', 'default_value')

print(my_dict)
Output
Click on try it to see output

If 'key4' is not present, it adds the key with the default value.

Choose the appropriate method based on your specific requirements for updating the dictionary.


Deleting Dictionary Elements

In Python, you can delete elements from a dictionary using several methods, depending on whether you want to delete a specific key, remove all items, or clear the entire dictionary. Here are the common methods.


Using the del Statement

The del statement is used to delete a specific key-value pair from a dictionary.

Example 
my_dict = {'key1': 'value1', 'key2': 'value2', 'key3': 'value3'}

del my_dict['key2']
print(my_dict)
Output
Click on try it to see output

pop() Method

The pop() method removes and returns the value associated with the specified key. If the key is not found, you can provide a default value to avoid a KeyError.

Example 
my_dict = {'key1': 'value1', 'key2': 'value2', 'key3': 'value3'}

value_of_key2 = my_dict.pop('key2', None)
print(my_dict)
Output
Click on try it to see output

popitem() Method

The popitem() method removes and returns the last key-value pair from the dictionary. This method is useful when you want to remove an arbitrary item.

Example 
my_dict = {'key1': 'value1', 'key2': 'value2', 'key3': 'value3'}

removed_item = my_dict.popitem()
print(my_dict)
print(f'Removed item: {removed_item}')

clear() Method

The clear() method removes all items from the dictionary, leaving it empty.

Example 
my_dict = {'key1': 'value1', 'key2': 'value2', 'key3': 'value3'}

my_dict.clear()
print(my_dict)
