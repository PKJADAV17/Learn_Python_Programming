datetime Class

The datetime class in Python is part of the datetime module and is used to represent points in time. It's a powerful and versatile class that includes both date and time information.


Creating datetime Objects

You can create a datetime object using the datetime() constructor. The parameters for the constructor include year, month, day, hour, minute, second, and microsecond. All parameters are optional, and if not provided, they default to 0.

Example 
from datetime import datetime

current_datetime = datetime.now()
print(current_datetime)

specific_datetime = datetime(2023, 1, 1, 12, 30, 0)
print(specific_datetime)

Attributes.

  • year: Year as a four-digit number.
  • month: Month as an integer (1-12).
  • day: Day of the month as an integer (1-31).
  • hour: Hour as an integer (0-23).
  • minute: Minute as an integer (0-59).
  • second: Second as an integer (0-59).
  • microsecond: Microsecond as an integer (0-999999).

strftime(format) Method

Returns a string representing the date and time according to a format string. Format codes are used to specify the representation.

Example 
from datetime import datetime

current_datetime = datetime.now()
formatted_date = current_datetime.strftime("%Y-%m-%d %H:%M:%S")
print("Formatted Date:", formatted_date)

strptime(date_string, format) Method

Parses a string representing a date and time according to a format string and returns a datetime object.

Example

parsed_date = datetime.strptime("2023-01-01 12:30:00", "%Y-%m-%d %H:%M:%S")

Date Class

In Python, the datetime module provides the date class, which represents a date (year, month, and day) without the time information. The date class is part of the broader datetime module, which also includes classes for handling time and combined date and time values.


Creating a date object and Access Values

You can create a date object using the date class constructor by specifying the year, month, and day.

You can access the year, month, and day components of a date object using the corresponding attributes.

Example 
from datetime import date

my_date = date(2023, 12, 1)
print(my_date)
print(my_date.year)   
print(my_date.month)  
print(my_date.day)

Arithmetic with dates

You can perform arithmetic operations with date objects, such as finding the difference between two dates.

Example 
from datetime import date

date1 = date(2023, 12, 1)
date2 = date(2023, 11, 1)
difference = date1 - date2
print(difference.days)
Output
Click on try it to see output

The date class does not represent time or timezone information. If you need to work with time, you can use the datetime class.

The date class is immutable, meaning you cannot modify its attributes once it's created. If you need to perform operations that involve modification, you can use the datetime class and then extract the date part.


Time Class

In Python, the datetime module provides the time class, which represents the time of day with hour, minute, second, and microsecond precision. The time class is part of the broader datetime module, which also includes classes for handling dates and combined date and time values.


Creating a time object and Access Values

You can create a time object using the time class constructor by specifying the hour, minute, second, and microsecond.

You can access the hour, minute, second, and microsecond components of a time object using the corresponding attributes.

Example 
from datetime import time

my_time = time(10, 30, 45, 123456)
print(my_time)
print(my_time.hour)         
print(my_time.minute)       
print(my_time.second)

Arithmetic with times

You can perform arithmetic operations with time objects, such as finding the difference between two times. However, note that the result might exceed 24 hours.

Example 
from datetime import timedelta, time

time1 = time(10, 30, 45, 123456)
time2 = time(8, 15, 30, 987654)
time_difference = timedelta(hours=time1.hour - time2.hour,
                            minutes=time1.minute - time2.minute,
                            seconds=time1.second - time2.second,
                            microseconds=time1.microsecond - time2.microsecond)
print(time_difference)

The time class does not represent date information. If you need to work with both date and time, you can use the datetime class.

The time class is immutable, meaning you cannot modify its attributes once it's created.


timedelta Class

In Python, the timedelta class is part of the datetime module and represents a duration of time. It is used for arithmetic operations with datetime, date, and time objects. timedelta can be used to represent the difference between two dates, times, or combined date and time values.


Creating a timedelta object and Access Values

You can create a timedelta object by specifying the duration in terms of days, hours, minutes, seconds, and microseconds.

You can access the individual components of a timedelta object, such as days, seconds, and microseconds.

Example 
from datetime import timedelta

delta = timedelta(days=5, hours=3, minutes=30)
print(delta)
print(delta.days)      
print(delta.seconds)   
print(delta.microseconds)

Arithmetic with timedelta

You can perform arithmetic operations with timedelta objects, such as addition or subtraction with datetime, date, or time objects.

Example 
from datetime import datetime, timedelta

current_datetime = datetime.now()

new_datetime = current_datetime + timedelta(days=2, hours=3)
print(new_datetime)

The timedelta class is immutable, meaning you cannot modify its attributes once it's created.
