Creating a Module

Creating a module in Python is a straightforward process.

  • Open your text editor or integrated development environment (IDE).
  • Create a new file and give it a name with a .py extension. For example, let's call it mymodule.py.
  • Add your Python code to the module file. This can include functions, classes, and variables.
  • You can now use this module in another Python script or interactive session.

Here's an example.

Example mymodule.py 
def greet(name):
    print(f"Hello, {name}!")

def add(a, b):
    return a + b

In this example, mymodule.py defines two functions (greet and add).


Example main_script.py 
import mymodule

mymodule.greet("Alice")
result = mymodule.add(3, 5)
print(f"The result is: {result}")

In this example, main_script.py imports the mymodule module and uses its greet and add functions.

This demonstrates the basic process of creating a module and using it in another script. Modules are a powerful way to organize and reuse code in Python projects.


Import Statement

In Python, the import statement is used to bring external modules or packages into your script or program. It allows you to access the functions, classes, and variables defined in the imported module. Here are different ways to use the import statement.


Importing the Entire Module

You can import the entire module and use its functions or classes using the module name followed by a dot (.).

Example 
import module_name

module_name.function_name()

Example 
import math

result = math.sqrt(25)
print(result)

Importing Specific Items from a Module

You can import specific functions or classes from a module instead of importing the entire module. This can make your code more concise.

Example 
from module_name import function_name

function_name()

Example 
from math import sqrt

result = sqrt(25)
print(result)

Importing with an Alias

You can use an alias to rename the imported module or item, providing a shorter or more convenient name.

Example 
import module_name as alias

alias.function_name()

Example 
import math as m

result = m.sqrt(25)
print(result)

Importing Everything from a Module

You can import all functions and variables from a module using the * wildcard. However, it is generally not recommended as it can lead to naming conflicts.

Example 
from module_name import *

function_name()

Example 
from math import *

result = sqrt(25)
print(result)

Note: Using * is discouraged because it can make the code less readable and may lead to unexpected behavior due to naming conflicts.


Importing Multiple Modules

In Python, you can import multiple modules in a single script by using multiple import statements. Each import statement brings in the functionality of a specific module, allowing you to use its functions, classes, and variables in your script.

Here's an example of how you can import multiple modules.

Example 
import math
import random

result_sqrt = math.sqrt(25)
print(f"Square root: {result_sqrt}")

random_number = random.randint(1, 10)
print(f"Random number: {random_number}")

from.. import Statement

The from ... import ... statement in Python is used to selectively import specific attributes (functions, classes, or variables) from a module into your script or another module. This syntax allows you to use the imported attributes directly without needing to prefix them with the module name.

Here's an example using the math module.

Example 
from math import sqrt, factorial

result_sqrt = sqrt(25)
result_factorial = factorial(5)

print(f"Square root: {result_sqrt}")
print(f"Factorial: {result_factorial}")

In this example, only the sqrt and factorial functions are imported from the math module, and they can be used directly without referencing the module name.
