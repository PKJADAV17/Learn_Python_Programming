Class Definition

You define a class using the class keyword, followed by the class name. The class body contains attributes and methods.

Example 
class MyClass:
    # class body

Attributes

Attributes are variables that store data associated with the class. They represent the characteristics or properties of objects created from the class.

Example 
class Dog:
    species = "Canis familiaris"  # Class attribute

    def __init__(self, name, age):
        self.name = name  # Instance attribute
        self.age = age    # Instance attribute

The __init__ method (constructor) is a special method used for initializing object attributes when an object is created.


Methods

Methods are functions defined inside a class. They perform operations related to the class and can access and modify the class attributes.

Example 
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name} says Woof!")

Instance and Instantiation

An instance is an individual object created from a class. Instantiation involves creating an instance of a class.

You create instances by calling the class as if it were a function, passing any required arguments to the __init__ method.

Example 
my_dog = Dog(name="Buddy", age=3)

Self

The self keyword is a reference to the instance of the class. It is the first parameter of every instance method and is used to access the attributes and methods of the instance.

Example 
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name} says Woof!")

In this example the bark method, self.name and self.age refer to the attributes of the instance.


Here is the simple example of a Person class with a couple of attributes (name and age) and a method to display information about the person.

Example 
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def display_info(self):
        print(f"Name: {self.name}")
        print(f"Age: {self.age}")

person1 = Person("Alice", 25)
person2 = Person("Bob", 30)
person1.display_info()
person2.display_info()

  • The display_info method prints information about the person.

  • We then call the display_info method for each person to display their information.

This is a basic example, but it illustrates the concept of classes, instances, attributes, and methods in Python. Classes provide a way to encapsulate related data and behavior, making the code more organized and reusable.
