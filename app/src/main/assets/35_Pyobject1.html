Class

A class is a blueprint for creating objects. It defines a data structure (attributes or properties) and methods (functions) that operate on that data. When you create an object, you are creating an instance of a class.

Example 
class Car:
    def __init__(self, make, model):
        self.make = make
        self.model = model

    def display_info(self):
        print(f"{self.make} {self.model}")

Attributes are variables that store data within an object. In the Car class example, make and model are attributes.

Methods are functions that are defined within a class and operate on the object's data. The display_info method in the Car class is an example.


Object

An object is an instance of a class. It is a concrete realization of the class blueprint, with its own unique data and behavior.

Example 
my_car = Car("Toyota", "Camry")
my_car.display_info()

In this example, my_car is an object of the Car class.


Here is the simple example using a Person class. This class will have attributes such as name and age, and a method to display information about the person.

Example 
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def display_info(self):
        print(f"Name: {self.name}, Age: {self.age}")

person1 = Person("John Doe", 25)

print(person1.name)  
print(person1.age)   
person1.display_info()

  • In this example We define a Person class with an __init__ method to initialize the object with a name and age.
  • The display_info method prints information about the person.
  • We create an instance of the Person class called person1 with the name "John Doe" and age 25.
  • We access the attributes (name and age) of the person1 object and call the display_info method to display information about the person.

This demonstrates the basic usage of a class, object, attributes, and methods in Python. You can expand on this example by adding more attributes or methods, or by creating other classes that interact with the Person class, demonstrating concepts like inheritance and polymorphism.
