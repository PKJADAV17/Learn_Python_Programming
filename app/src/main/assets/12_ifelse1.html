if statement

The if statement is a fundamental control flow statement in Python that allows you to execute a block of code conditionally.

The basic syntax of the if statement is as follows.

Syntax 
if condition:
    # Code to be executed if the condition is true

Here's a breakdown of the components.

  • if: This is the keyword that starts the if statement.
  • condition: This is an expression that evaluates to either True or False. If the condition is True, the indented block of code beneath it will be executed. If the condition is False, the code block is skipped.
  • : (colon): It signifies the end of the if statement's header and the beginning of the indented block.

Here's a simple example.

Example 
x = 10

if x > 5:
    print("x is greater than 5")

In this example, the condition x > 5 is True, so the indented block beneath the if statement is executed, and "x is greater than 5" is printed.


if-else statement

The if-else statement is used in Python for branching or conditional execution of code. It allows you to specify two blocks of code: one to be executed if a certain condition is true (if block), and another to be executed if the condition is false (else block).

The basic syntax of the if-else statement is as follows.

Syntax 
if condition:
    # Code to be executed if the condition is true
else:
    # Code to be executed if the condition is false

Here's a simple example.

Example 
x = 10

if x > 5:
    print("x is greater than 5")
else:
    print("x is not greater than 5")
Output
Click on try it to see output

In this example, if the condition x > 5 is true, the code block under the if statement is executed, and "x is greater than 5" is printed. If the condition is false, the code block under the else statement is executed, and "x is not greater than 5" is printed.


if-elif-else statement

The if-elif-else statement in Python is an extension of the basic if-else statement and is used when you have multiple conditions to check. It allows you to provide different code blocks to be executed based on which condition is true first.

The basic syntax is as follows.

Syntax 
if condition1:
    # Code to be executed if condition1 is true
elif condition2:
    # Code to be executed if condition2 is true
elif condition3:
    # Code to be executed if condition3 is true
    # ...
else:
    # Code to be executed if none of the conditions is true

Each elif (short for "else if") introduces a new condition to be checked, and the code block under the first true condition is executed. If none of the conditions is true, the code block under the else statement is executed.


Here's an example.

Example 
x = 5

if x > 5:
    print("x is greater than 5")
elif x < 5:
    print("x is less than 5")
else:
    print("x is equal to 5")
Output
Click on try it to see output

In this example, the program first checks if x is greater than 5. Since it's not, it moves on to the next condition, checking if x is less than 5. Finally, if neither of the conditions is true, it executes the code block under the else statement.


Nested if statements

Nested if statements in Python allow you to have one or more if statements inside another. This enables you to create more complex decision-making logic and handle multiple conditions in a structured manner.

The basic syntax for nested if statements is as follows.

Syntax 
if outer_condition:
    # Outer code block
    if inner_condition:
        # Inner code block
    else:
        # Another inner code block
else:
# Code block if outer_condition is false

Here's a simple example.

Example 
x = 10
y = 5

if x > 5:
    print("x is greater than 5")
    if y > 2:
        print("y is greater than 2")
    else:
        print("y is not greater than 2")
else:
    print("x is not greater than 5")
Output
Click on try it to see output

In this example, the outer if statement checks if x is greater than 5. If this condition is true, it enters the indented block and further checks if y is greater than 2 using the inner if statement. Depending on the values of x and y, different code blocks will be executed.


Ternary conditional expression

The ternary conditional expression, also known as the ternary operator or conditional expression, is a concise way to write a simple if-else statement in a single line. It provides a compact syntax for conditional assignments or expressions.

The basic syntax is as follows.

Syntax 
value_if_true if condition else value_if_false

Here's a simple example.

Example 
x = 10
result = "greater than 5" if x > 5 else "not greater than 5"
print(result)
Output
Click on try it to see output

In this example, if the condition x > 5 is true, the value "greater than 5" is assigned to the variable result. If the condition is false, the value "not greater than 5" is assigned.

The ternary conditional expression can be used for various purposes, such as concise assignments or creating short conditional expressions within larger expressions. It is especially useful when the if-else statement is straightforward and involves simple expressions.
