Variable Naming Rules
  • Variable names must begin with a letter (a-z, A-Z) or an underscore (_).
  • The subsequent characters can be letters, numbers, or underscores.
  • Variable names are case-sensitive (e.g., myVar and myvar are different).
  • Python keywords (reserved words) cannot be used as variable names.

Variable Declaration and Assignment

In Python, you don't explicitly declare variables in the way you might in some other programming languages (e.g., C or Java). Instead, you simply assign a value to a variable, and Python dynamically determines the variable's type based on the assigned value.

Here's an example.

Example 
# Variable declaration and assignment
x = 10
name = "John"
pi = 3.14

# Displaying the values
print(x)    # Output: 10
print(name) # Output: John
print(pi)   # Output: 3.14

In this example, x, name, and pi are variables that are assigned values of different types (integer, string, and float, respectively). Python infers the types automatically.


It's important to note that Python is a dynamically-typed language, which means you can reassign a variable to a value of a different type

Example 
x = 10
print(x)  # Output: 10

x = "Hello"
print(x)  # Output: Hello
Output
Click on try it to see output

In this example, x is initially assigned the value 10, but later it's reassigned the value "Hello".

While Python doesn't have variable declaration in the traditional sense, you can think of variable assignment as a way of declaring and initializing variables simultaneously. The lack of explicit declaration contributes to the simplicity and flexibility of Python code.


Dynamic Typing

Python is dynamically typed, meaning you don't have to explicitly declare the type of a variable. The interpreter infers the type based on the assigned value.

Example 
x = 10        # x is an integer
y = "hello"   # y is a string

Data Types

Variables in Python can hold values of different data types, such as integers, floats, strings, lists, tuples, dictionaries, etc.

Example 
num = 42      # integer
pi = 3.14     # float
message = "Hello, World!"  # string

Variable Scope

The scope of a variable determines where it can be accessed. Variables can have local or global scope.

  • Local Scope: Variables defined within a function have local scope and are only accessible within that function.
  • Global Scope: Variables defined outside of any function have global scope and can be accessed throughout the program.
Example 
global_var = 10  # Global variable

def my_function():
    local_var = 5  # Local variable

Deleting Variables

You can use the del statement to delete a variable and free up the memory it occupies.

Example 
x = 10
del x

These are the basic concepts of Python variables. Understanding how to use variables effectively is crucial for writing clear, concise, and functional code.
