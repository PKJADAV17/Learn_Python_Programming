String Slicing

String slicing is the process of creating a substring (a portion of a string) by extracting a sequence of characters from the original string. The basic syntax for string slicing in Python is as follows.

Syntax 
string[start:stop:step]
  • start: The index of the first character to include in the substring. It is inclusive, meaning the character at this index is part of the substring.
  • stop: The index of the first character to exclude from the substring. The character at this index is not included in the substring. The substring includes characters up to, but not including, the character at the stop index.
  • step: The step or interval between characters. It determines how many characters to skip between each inclusion. The default value is 1.

Here are some examples to illustrate string slicing.

Example 
my_string = "Hello, World!"

substring1 = my_string[7:12]  
print(substring1)

substring2 = my_string[:5]    
print(substring2)

substring3 = my_string[7:]    
print(substring3)

substring4 = my_string[-6:-1] 
print(substring4)

substring5 = my_string[::2]   
print(substring5)
Output
Click on try it to see output

In this example

  • substring1 extracts the characters from index 7 to index 11.
  • substring2 includes characters from the beginning of the string up to index 4.
  • substring3 includes characters from index 7 to the end of the string.
  • substring4 uses negative indices to extract characters from the sixth-to-last to the second-to-last position.
  • substring5 uses a step of 2 to include every second character.

It's important to note that string slicing does not modify the original string; it creates a new string containing the specified substring. Strings in Python are immutable, meaning their contents cannot be changed after creation.

String slicing is a powerful feature that allows you to work with substrings efficiently, making it easier to manipulate and extract information from strings in your programs.


String Formatting

Python supports different ways of formatting strings, such as the % operator, the format() method, and f-strings.


% Operator

You can use the % operator for string formatting, similar to how it's done in C's printf function.

Example 
name = 'Alice'
age = 30
formatted_string = 'My name is %s and I am %d years old.' % (name, age)
print(formatted_string)

  • %s is a placeholder for a string.
  • %d is a placeholder for an integer.

f-strings (Python 3.6 and later)

f-strings provide a concise and readable way to format strings.

Example

age = 35
message = f"My name is {name} and I am {age} years old."
print(message)

String Methods

String methods in Python are built-in functions that operate on strings and allow you to perform various operations such as searching, modifying, and formatting.

Here are some commonly used string methods.

  • len(): Returns the length of the string.
  • upper(): Converts all characters in the string to uppercase.
  • lower(): Converts all characters in the string to lowercase.
  • capitalize(): Converts the first character to uppercase and the rest to lowercase.
  • title(): Converts the first character of each word to uppercase.
  • strip(): Removes leading and trailing whitespaces from the string.
  • replace(old, new): Replaces occurrences of a substring (old) with another substring (new).
  • find(substring): Returns the index of the first occurrence of the specified substring. Returns -1 if the substring is not found.
  • count(substring): Returns the number of occurrences of the specified substring in the string.
  • startswith(prefix), endswith(suffix): Checks if the string starts or ends with the specified prefix or suffix, respectively. Returns a Boolean.
  • split(separator): Splits the string into a list of substrings based on the specified separator.
  • join(iterable): Joins the elements of an iterable (e.g., a list) into a string using the current string as a separator.
Example 
my_string = "Hello, World!"
length = len(my_string)  
print(length)

upper_case = my_string.upper() 
print(upper_case)

lower_case = my_string.lower() 
print(lower_case)

capitalized = my_string.capitalize() 
print(capitalized)

titled = my_string.title()
print(titled)

stripped = my_string.strip()
print(stripped)

index = my_string.find("World")
print(index)

starts_with_hello = my_string.startswith("Hello")  
print(starts_with_hello)

ends_with_world = my_string.endswith("World") 
print(ends_with_world)

new_string = my_string.replace("Hello", "Hi")
print(new_string)
